► Chrome-расширение: метки юзерам Pikabu


Серьёзный недостаток заметок на Пикабу — их нет перед глазами. Метки этого расширения всегда на виду, справа от никнейма или под ним — в профиле, постах и комментариях.

Реализовано на чистом JS + CSS, иконки Font Awesome. Метки хранятся на стороне пользователя в chrome.storage.sync (по документации — с автосинхронизацией между браузерами одной учётки Google, но, судя по всему, заработает только после публикации в Chrome Web Store).

► Переделка таксофона АМТ-69 под сотовую связь


С помощью Arduino Nano и модуля SIM900 таксофону АМТ-69 была подарена новая жизнь. Старался сделать всё максимально антуражно:

  • при входящем вызове звучит «правильный» звонок — тарахтение по колокольчикам;
  • ответ на звонок поднятием трубки, завершение — опусканием на рычаг;
  • входящие и исходящие вызовы работают без нареканий;
  • номера экстренных служб в старом формате (01, 02…) автоматически преобразуются в новый (101, 102…);
  • при поднятии трубки звучит гудок, при наборе — DTMF-сигналы, при окончании — короткие гудки (по стандартам);
  • работа от АКБ 12В.

От монетоприёмника отказался: громоздкий механизм, нет оригинальных жетонов, нет времени переделывать под актуальные деньги.

► Реверс-инжиниринг камеры дрона DJI Phantom FC40 [I H4CK3D A DR0NE!!1]

Для работы с камерой было доступно лишь приложение для смартфона, которое уже устарело и не работает на актуальных мобильных ОС.

Старым смартфоном отловил трафик общения приложения с камерой, анализ — в Wireshark. Протокол не был защищён: набор эндпоинтов, на которые отправлялись HTTP-запросы с командами. Написал скрипт на Python, который инициализировал работу камеры и открывал RTSP-поток для любой программы с поддержкой потокового видео. Скрипт слал статусные сообщения (клиент не отключился) и выводил уровень заряда аккумулятора камеры.
